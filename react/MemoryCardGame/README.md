# Memory Card Game

A simple and customizable **Memory Card Game** built with **React**.

This project demonstrates a classic memory matching game where cards can contain virtually any type of content, making it easy to create different themes and game variations.

## ✨ Features

- ⚛️ Built with React
- 🎴 Reusable and customizable card components
- 🔀 Randomized card layout on each game
- ✅ Match detection logic

## 🛠️ Tech Stack

- React
- JavaScript
- CSS
